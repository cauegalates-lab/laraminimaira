LARAMINIMARIA - SITE DE PERFUMES

Arquivos:
- index.html: página completa da loja.
- imagens/: pasta para colocar a logo e as fotos dos perfumes.

Como editar:
1. Abra o arquivo index.html no VS Code.
2. Procure por: const WHATSAPP_NUMBER = "5512999999999";
3. Troque pelo WhatsApp real, usando DDI + DDD + número. Exemplo: 5512999999999.
4. Procure por: const produtos = [
5. Altere nome, descrição, preço, categoria, volume e imagem dos produtos.
6. Coloque as imagens reais dos perfumes dentro da pasta imagens.
7. Para trocar a logo do topo, substitua o arquivo imagens/logo-header.png mantendo o mesmo nome.
8. No campo imagem, use o caminho assim: imagens/nome-da-imagem.png

Exemplo:
imagem: "imagens/floral-rosa.png"

Para testar:
- Dê dois cliques no index.html.
- Ou abra pelo VS Code com Live Server.
