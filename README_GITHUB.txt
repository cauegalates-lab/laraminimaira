INSTRUÇÕES PARA SUBIR NO GITHUB PAGES

1. Abra esta pasta e envie para o repositório os arquivos que estão aqui dentro:
   - index.html
   - pasta imagens
   - favicon.ico
   - favicon.png

2. A pasta imagens precisa ficar no mesmo nível do index.html.

Estrutura correta:

index.html
imagens/
  logo-header.png
  logo-laraminimaria.png
  favicon.png
  card-femininos.png
  card-masculinos.png
  ...

3. Não envie apenas o index.html. Se a pasta imagens não subir junto, as logos e fotos não aparecem.

4. Os caminhos das imagens já foram ajustados para GitHub Pages usando ./imagens/nome-do-arquivo.png.
